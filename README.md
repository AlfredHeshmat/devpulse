# DevPulse

DevPulse is a GitHub Analytics Dashboard built with React and the GitHub REST API. It allows users to search any GitHub profile and view developer insights, repository statistics, and top projects.

## Features

- Search GitHub users by username
- View profile information
- Display followers, following, and public repository count
- Analyze repositories
- Calculate total stars
- Identify the most used programming language
- Display the top repository
- Show top 5 repositories sorted by stars
- Show repository update dates
- Open GitHub profiles and repositories directly
- Loading and error handling
- Responsive dark UI

## Technologies Used

- React
- JavaScript
- Vite
- CSS
- GitHub REST API
